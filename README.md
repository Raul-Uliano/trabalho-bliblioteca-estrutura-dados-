# Projeto Final - Sistema de Biblioteca

Projeto desenvolvido para a atividade avaliativa de Estrutura de Dados.

## Cenário escolhido

Sistema de Biblioteca.

## Objetivo

Criar uma arquitetura simples para controlar livros, usuários, empréstimos, fila de espera e ranking dos livros mais emprestados.

## Estruturas de dados utilizadas

- Map para armazenar livros
- Map para armazenar usuários
- Array para armazenar empréstimos ativos
- Fila para controlar a espera por livros indisponíveis
- Array ordenado para gerar o ranking dos livros mais emprestados

## Arquivos do projeto

- `index.js`: código do sistema
- `diagrama.md`: diagrama UML simples
